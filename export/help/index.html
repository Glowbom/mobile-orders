  • Mobile Orders

    Create your own mobile ordering app without code.

    Using a Google Sheet.



    This app template can help business owners get set up a mobile ordering app.


    Please follow the instructions below:

  • Step 1

    Open the Google Sheet Template:

    Open Template

  • Step 2

    Make a copy of the Google Sheet Template:


  • Step 3

    Put your products into the sheet.


    The media column supports any links to images, webpages, and youtube videos.

  • Step 4

    Open Script editor from Tools menu.

  • Step 5

    Download the Backend Script file:

    Get Script

    Copy backendscript.txt file content, paste it to Code.gs file inside the Script editor, and save changes.

  • Step 6

    Go to Publish -> Deploy as web app...

  • Step 7

    Change Who has access to the app to Anyone, even anonymous.


    Click Deploy button.

  • Step 8

    Click Review Permissions and give permissions to your backend script.

  • Step 9

    Copy Current web app URL.

  • Step 10

    Donwload the source code:

    Get App Code

  • Step 11

    Unzip the code, open the iOS project in Xcode, and put Current web app URL inside the line 13 of NetworkManager.swift file.


    Open the Android project in Android Studio, and put Current web app URL without https://script.google.com/ inside the line 13 of ItemsApi.kt file.

  • Step 12

    Run the app and place the order.

    You will be able to see the order inside the Orders tab of your Google Sheet.
